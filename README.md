# meemow-companion
its a cute little interactive robot (a tamagochi like) but noonafters version specifically for my little sister!

# what meemow does! V1
for now im going to keep it simple, meemow will be able to say hello, says it hungry when its been on for a long time and will be fed, and say thankyou with arms moving up and down because it happy that its fed
basically:
- say hello (greeting expression)
- says hungry and will be fed (hungry expression, feeding expression + feeding button)
- says thankyou (happy expression + arms moving hapily)
